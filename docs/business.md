# Business — Diretriz Tecnologia

> Site institucional de divulgação de marca e captação de leads.
> Domínio: **diretriztecnologia.com.br**
> Idiomas: **PT-BR** (primário) + **EN** (secundário).

---

## 1. Resumo executivo

A **Diretriz Tecnologia** entrega soluções de software sob medida para empresas que precisam **desenvolver um produto novo** ou **automatizar fluxos internos** sem trocar os sistemas que já usam.

O site institucional existe para **três objetivos** nesta ordem de prioridade:

1. **Captar leads qualificados** via página de contato (form + e-mail estruturado).
2. **Posicionar a marca** como parceira tecnológica moderna, robusta e confiável.
3. **Educar o visitante** sobre o que a Diretriz faz, com foco em IA aplicada a negócios.

A v1 NÃO é um catálogo de cases nem blog. É uma **carta de apresentação interativa** — densidade visual alta, densidade textual baixa.

## 3. Público-alvo

### Persona primária — "O Operador Pressionado"

- Diretor de operações ou CTO em empresa de **médio porte** (50–500 funcionários).
- Tem um sistema interno que **funciona, mas trava o crescimento**: planilhas, ERPs antigos, processos manuais.
- Quer **automatizar e modernizar sem rasgar tudo**.
- Decisor ou influenciador direto de compra.
- Critério de escolha: **rapidez + previsibilidade**, não menor preço.

### Persona secundária — "O Empreendedor com Ideia"

- Founder/sócio em estágio inicial ou empresa estabelecida com **ideia de produto novo**.
- Tem o problema mapeado, falta o time técnico para construir.
- Procura **parceiro de execução**, não freelancer.
- Critério de escolha: **expertise + apresentação profissional**, ticket aceita ser maior.

### Persona terciária — "O Curioso Técnico"

- Desenvolvedor, arquiteto ou recrutador navegando o mercado.
- Não vai virar cliente, mas pode virar **canal de indicação ou contratação**.
- O site precisa **respeitar o olhar técnico** (código limpo no source, performance boa, sem cringe).

## 4. Pilares de oferta

A v1 do site comunica quatro pilares de serviço. **Sem preço**, sem catálogo — apenas escopo qualitativo.

### 4.1 Produtos sob medida
Construção de software do zero para empresas que querem **lançar um produto próprio**: web, mobile, integrações.

### 4.2 Automação de fluxos internos
Mapear processos manuais e substituir por workflows automatizados, sem trocar os sistemas existentes.

### 4.3 Integração com IA
Adicionar **camada de IA sobre o sistema legado** do cliente: chat sobre os dados dele, classificação de documentos, copilot dentro da ferramenta atual, RAG sobre base de conhecimento. **Posicionamento-chave:** não pedimos para o cliente trocar de sistema, multiplicamos o valor do que ele já tem.

### 4.4 Consultoria técnica
Discovery, arquitetura, code review, segunda opinião sobre stack — para empresas com time interno que precisam de validação externa.

## 6. Jornada do visitante e funil de lead

```
   Aquisição                Engajamento                    Conversão
─────────────────       ──────────────────────       ─────────────────────
Google (SEO)        →   Hero (impacto visual)    →   CTA Header sticky
Indicação           →   Pilares (3 scrolls)      →   Form de contato
LinkedIn / direto   →   Seção IA (diferencial)   →   E-mail estruturado
                        Sobre + processo             para a Diretriz
                        Casos / clientes (v2)
```

### Estados-meta de cada visitante

1. **Visitante frio (60%)** — chega via Google ou redes. Sai sem agir. Site precisa cravar marca para retornar depois.
2. **Visitante morno (30%)** — entendeu a oferta, lê toda a página, sai. Próxima visita pode converter.
3. **Visitante quente (10%)** — chega com intenção, vai direto ao contato. **Form precisa ser frictionless.**

### Form de contato (campos)

- Nome\*
- Empresa
- E-mail\*
- Telefone (opcional)
- Tipo de interesse (radio: produto novo / automação / IA / consultoria / outro)
- Descrição livre do problema (textarea)
- Como nos achou? (opcional, ajuda atribuição)

Asterisco = obrigatório. Mínimo de campos para reduzir abandono.

## 7. Métricas de sucesso

### Métricas de site (mensais)
- Visitantes únicos
- Taxa de scroll até pilares (engajamento real)
- Taxa de conversão **visitante → form enviado** (alvo inicial: 1–3%)
- Tempo médio na página
- Origem de tráfego (orgânico vs direto vs social)

### Métricas de negócio (mensais)
- Leads recebidos pelo form
- Leads qualificados (pós-triagem da Diretriz)
- Reuniões agendadas
- Propostas enviadas
- **Conversão lead → cliente** (alvo de longo prazo)

Ferramenta: **Cloudflare Web Analytics** (sem cookie, sem banner LGPD, suficiente para v1).

## 8. Identidade visual

### Logo
Avião de papel estilizado em azul forte + cinza-azulado, wordmark "Diretriz" em azul-marinho com "tecnologia" em azul vibrante embaixo. Transmite **direção, movimento, precisão**.

### Paleta (extraída do logo + referências)

| Token | Hex aproximado | Uso |
|---|---|---|
| `--color-bg` | `#0A1628` | Background principal (dark) |
| `--color-bg-elevated` | `#0F1E36` | Cards, surfaces |
| `--color-primary` | `#1D4ED8` | Acentos, CTAs, links |
| `--color-primary-bright` | `#3B82F6` | Hover, glow, gradientes |
| `--color-accent` | `#22D3EE` | Detalhes, IA, "vivacidade" |
| `--color-neutral` | `#64748B` | Texto secundário, separadores |
| `--color-text` | `#F1F5F9` | Texto principal sobre dark |
| `--color-text-muted` | `#94A3B8` | Texto auxiliar |

> Cores definitivas serão validadas no protótipo. Estes são pontos de partida derivados do logo.

### Tipografia (definida)
- **Display + Body**: **Geist Sans** (família unificada da Vercel, free)
  - Pesos: 400 (body), 500 (subtle emphasis), 600 (headings), 700 (display grande)
- **Mono** (acentos técnicos, números, tags): **Geist Mono** — peso 500
- Tamanhos via escala fluida com `clamp()`
- Carregamento via `@font-face` self-hosted (não Google Fonts) para performance

### Inspirações de execução
- **Infinite Field** — brutalismo-tech, scroll cinemático, hero único forte
- **Autonomous Finances** — 3D embutido, paleta ciano sobre dark, microinterações ricas
- **Kriss.ai** — preloader cinemático, gradientes etéreos, sensação de "produto IA"

## 10. Roadmap

### v1 — Lançamento (escopo deste planejamento)
- 4 páginas (home, sobre, serviços, contato) em PT + EN
- Hero 3D / WebGL na home
- Form de contato funcional via EmailJS
- SEO básico (meta tags, sitemap, robots, Open Graph)
- Analytics (Cloudflare Web Analytics)
- Responsivo mobile/desktop
- Performance LCP < 2.5s, CLS < 0.1

### v2 — Crescimento (3–6 meses pós-lançamento)
- Chatbot de qualificação de lead (IA — backend em Cloudflare Worker)
- Página de cases (após primeiros clientes autorizarem)
- Blog técnico em markdown (Astro Content Collections)
- A/B test no hero
- Otimização SEO baseada em search console

### v3 — Escala
- Demo interativa de IA pública (ferramenta gratuita como ímã de lead)
- Calculadora de orçamento estimado
- Integração com CRM (Pipedrive / HubSpot)
- Newsletter
